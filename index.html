Project Overview

This project implements an intelligent embedded HMI for a simulated four-cell battery system. The interface automatically rotates between multiple diagnostic screens showing real-time voltage readings, pack analytics, protection status, fault diagnostics, state of charge, health status, relay status, and active faults.

The system is designed with optimized LCD refresh logic to avoid flicker. During critical conditions, the normal rotating dashboard is immediately overridden by a fault-priority screen so that the most important warning is shown first.

HMI Screen Rotation

The HMI automatically rotates between multiple diagnostic screens during normal operation. Each screen provides a specific category of information to make the system easier to monitor and debug.

Screen 1

Live cell voltages for Cell 1, Cell 2, Cell 3, and Cell 4.

Screen 2

Pack voltage and average battery cell voltage.

Screen 3

State of charge and imbalance percentage.

Screen 4

System state and relay protection status.

Screen 5

Strongest cell, weakest cell, and active fault count.

Screen 6

Battery health status such as Excellent, Good, Warning, or Critical.

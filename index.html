Project Overview

This project implements an intelligent embedded HMI for a simulated four-cell battery system. The interface automatically rotates between multiple diagnostic screens showing real-time voltage readings, pack analytics, protection status, fault diagnostics, state of charge, health status, relay status, and active faults.

The system is designed with optimized LCD refresh logic to avoid flicker. During critical conditions, the normal rotating dashboard is immediately overridden by a fault-priority screen so that the most important warning is shown first.

HMI Screen Rotation

The HMI automatically rotates between multiple diagnostic screens during normal operation. Each screen provides a specific category of information to make the system easier to monitor and debug.

Screen 1

Live cell voltages for Cell 1, Cell 2, Cell 3, and Cell 4.

Screen 2

Pack voltage and average battery cell voltage.

Screen 3

State of charge and imbalance percentage.

Screen 4

System state and relay protection status.

Screen 5

Strongest cell, weakest cell, and active fault count.

Screen 6

Battery health status such as Excellent, Good, Warning, or Critical.

Automatic Fault Override

Whenever a critical condition is detected, the rotating HMI immediately stops and displays the highest-priority warning screen. This ensures that operators always see the most important fault first.

Weak Cell

LCD immediately displays the affected battery cell and relay status.

Over Voltage

High-voltage conditions override every normal information screen.

Sensor Fault

Sensor failures are highlighted instantly to prevent invalid readings.

Imbalance Warning

Excessive imbalance automatically triggers diagnostic mode.

Relay Status

Relay ON/OFF state is continuously monitored and displayed.

Automatic Recovery

After faults clear, the interface automatically resumes normal rotating diagnostics.

Battery Analytics

Pack Voltage

Total voltage of all battery cells combined.

Average Voltage

Average voltage calculated across the battery pack.

Battery SOC

Estimated State of Charge calculated from measured voltages.

Cell Imbalance

Difference between strongest and weakest battery cell.

Strongest Cell

Automatically identifies the healthiest battery cell.

Weakest Cell

Detects the weakest battery cell for diagnostics.
